# CHChatKit
CHChatKit is an Chat Components and Instant Messaging library written in Object-C.

## 目前实现1.0功能
* 发送消息
* 发送图片
* 发送声音
* 本地数据库－文字
* 草稿

# 资源替换
聊天中的资源文件使用Bundle方式引入，如果需要替换，按照原先的Bundle图片名称替换到一个新建的Bundle中，Bundle的命名规范是这样的CHChatCustomized的文件在工程中.
